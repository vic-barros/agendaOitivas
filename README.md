\# 📅 Agenda de Oitivas

Projeto desenvolvido em Javafocado em lógica de programação e organização de fluxos. O projeto visa facilitar o agendamento e organização de oitivas policiais nas delegacias, diante da alta demanda nas unidade policiais, o projeto vem colaborar com a organização dos policiais civis.



\## 🚀 Tecnologias

\- Java (JDK 17+)

\- Git \& GitHub para versionamento



\## 📚 Contexto

Este projeto faz parte dos meus estudos no curso de\*Análise e Desenvolvimento de Sistemas (ADS), aplicando conceitos de Orientação a Objetos.
